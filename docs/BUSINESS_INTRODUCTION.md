# Hx.Abp.Attachment - 智能档案管理系统业务介绍

## 📋 项目概述

**Hx.Abp.Attachment** 是一款面向企业级应用的智能档案管理系统，致力于通过人工智能技术和现代化的架构设计，帮助企业实现档案管理的数字化转型，提升知识资产的管理效率和价值挖掘能力。

### 🎯 核心价值主张

#### 1. 智能化档案管理

-   **AI 驱动的自动分类**：基于自然语言处理技术，自动识别文档内容并推荐合适的分类
-   **语义搜索能力**：支持自然语言查询，理解用户意图，精准定位目标文档
-   **智能元数据提取**：自动提取文档关键信息，构建结构化知识库

#### 2. 标准化分类体系

-   **灵活的模板管理**：支持创建、版本管理、结构导出等多种模板操作
-   **动态/静态分面**：灵活支持固定分类和动态分类两种模式
-   **业务规则验证**：内置完善的业务规则验证，确保分类体系的规范性和一致性

#### 3. 企业级能力

-   **权限精细化管理**：支持多层级权限控制，满足企业安全合规要求
-   **版本追溯能力**：完整的版本管理机制，支持历史版本查看和回滚
-   **高性能架构**：基于 ABP Framework 和 PostgreSQL，支持大规模数据存储和查询

## 💼 核心业务功能

### 1. 智能分类模板管理

#### 功能概述

提供完整的分类模板管理能力，支持模板的创建、更新、版本管理、结构导出等全生命周期操作。

#### 核心能力

**模板创建与管理**

-   支持创建根分类模板和子分类模板
-   支持模板的版本管理，可以创建多个版本并指定最新版本
-   支持模板的树形结构管理，可以构建复杂的分类体系
-   支持动态分面和静态分面两种模式

**业务规则验证**

-   **根分类模板不能是动态分面**：确保根模板的稳定性，避免根分类频繁变化
-   **同一级只能有一个动态分面模板**：避免同级多个动态分面导致结构混乱
-   **动态分面和静态分面互斥**：同一级不能同时存在动态和静态分面模板，保持分类结构清晰
-   **模板名称唯一性验证**：同一父节点下模板名称不能重复，根节点下也不能重复

**模板结构导出**

-   支持将模板结构导出为 ZIP 压缩包
-   压缩包包含完整的目录结构
-   动态分面模板在名称后添加 `(动态分类)` 标记，便于识别
-   每个模板文件夹包含 `模板信息.txt` 文件，记录模板详细信息

#### 业务价值

-   **标准化管理**：通过模板管理，确保分类体系的标准化和一致性
-   **灵活扩展**：支持动态分面，适应业务变化需求
-   **知识沉淀**：模板结构导出功能，便于知识沉淀和分享

### 2. 智能文档分类

#### 功能概述

基于人工智能技术，自动识别文档内容并推荐合适的分类，支持批量处理和实时分类。

#### 核心能力

**AI 分类推荐**

-   集成阿里云 OpenNLU 服务，支持自然语言理解
-   基于文档内容自动推荐分类
-   提供分类置信度评分，帮助用户判断分类准确性
-   支持自定义分类体系训练

**批量处理能力**

-   支持大规模文档批量分类
-   异步处理机制，提升处理效率
-   批量处理进度跟踪和结果反馈

**多维度分类**

-   支持按业务、部门、项目等多维度分类
-   支持标签体系，便于文档标记和检索
-   支持元数据字段，丰富文档属性信息

#### 业务价值

-   **提升效率**：自动分类减少人工操作，提升分类效率 70%以上
-   **提高准确性**：AI 驱动的分类推荐，提高分类准确性
-   **降低成本**：减少人工分类成本，降低运营成本

### 3. 智能检索能力

#### 功能概述

提供全文搜索、语义搜索、关键词提取等多种检索能力，帮助用户快速定位目标文档。

#### 核心能力

**全文搜索**

-   基于 PostgreSQL 全文搜索能力
-   支持中文分词和模糊匹配
-   支持多字段组合搜索
-   搜索结果按相关性排序

**语义搜索**

-   基于向量相似度计算的语义搜索
-   支持自然语言查询，理解用户意图
-   支持多维度搜索排序
-   搜索结果优化和去重

**关键词提取**

-   基于 TF-IDF 和 TextRank 的关键词提取
-   自动识别文档核心词汇
-   支持关键词权重排序
-   支持自定义关键词词典

#### 业务价值

-   **提升检索效率**：智能检索能力，提升文档查找效率 60%以上
-   **改善用户体验**：自然语言查询，降低使用门槛
-   **发现知识关联**：语义搜索帮助发现文档之间的关联关系

### 4. 权限与安全管理

#### 功能概述

提供完善的权限管理能力，支持多层级权限控制，满足企业安全合规要求。

#### 核心能力

**多层级权限控制**

-   系统级权限：管理员权限
-   分类级权限：按档案分类控制访问
-   文档级权限：单个文档的细粒度权限
-   操作级权限：读、写、删除等操作权限

**权限配置管理**

-   支持基于角色的权限配置
-   支持自定义权限策略
-   支持权限继承和覆盖
-   支持权限审计和追溯

**安全审计**

-   完整的操作日志记录
-   支持操作追溯和审计
-   支持敏感操作告警
-   支持数据脱敏展示

#### 业务价值

-   **满足合规要求**：完善的权限管理和审计能力，满足企业合规要求
-   **保障数据安全**：多层级权限控制，保障数据安全
-   **降低安全风险**：权限审计和追溯，降低安全风险
